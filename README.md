# 🛒 Cart System

![HTML](https://img.shields.io/badge/HTML-5-orange)
![CSS](https://img.shields.io/badge/CSS-3-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Bootstrap](https://img.shields.io/badge/Bootstrap-5-purple)

A simple and powerful **Cart System** built with Vanilla JavaScript and Bootstrap.
This project simulates a real product management system with an Admin panel.

---

## 🚀 Live Demo

👉 https://eslam-mohamed-1892.github.io/Cart-System/

---

## ✨ Features

### 👨‍💼 Admin Panel

* ➕ Add new products using modal
* 🗑️ Delete products
* 🔄 Auto update UI
* 💾 Save data in Local Storage

### 🧑‍💻 User Page

* 👀 View all products
* 🔄 Sync with admin updates

---

## 🧠 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

---

## ⚙️ How It Works

1. Admin adds a product from modal
2. Data stored in Local Storage
3. Products rendered dynamically
4. User page reads same data
